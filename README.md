# ShadowLayout
### 带阴影的布局  
shadowColor:阴影颜色  
shadowRadius:阴影弧度  
shadowDx:阴影X轴偏移量  
shadowDy:阴影Y轴偏移量  
rectangle:矩形  
oval:椭圆形  
all:四边  
left:左边  
top:上边  
right:右边  
bottom:下边

    allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  
    dependencies {
	        implementation 'com.github.Wiser-Wong:ShadowLayout:1.0.1'
	}
